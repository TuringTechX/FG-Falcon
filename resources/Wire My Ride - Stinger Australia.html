Lets Get Started!

1. Configuring the interface for your vehicle

Before connecting or wiring your vehicle, the ControlPRO interface will need to know what vehicle it is being used with. To do this set the group of 8 DIP switches found on the left side of the interface. Use the settings shown below

Vehicle Setting

6

7

8

2. Configuring the interface for your aftermarket radio

The ControlPRO interface will need to know what aftermarket radio it is being used with. To do this set the group of 4 DIP switches found on the right side of the interface. Use the settings shown below.

3. Wiring your interface to your vehicle.

Please make sure the interface is unplugged from the supplied harness and vehicle.

ControlPRO Vehicle Wiring Information

Use the diagram, Pin locations and wire colours below to connect the ControlPRO harness to your vehicle and aftermarket radio. Any unused wires please terminate and tape up.

Please splice onto red/white wire at steering column. *** DO NOT CUT ***
ControlPRO Harness Black Wire:
CHASSIS GROUND in vehicle
ControlPRO Harness Red Wire:
ACC POWER in vehicle
ControlPRO Harness Yellow Wire:
CONSTANT 12v+ in vehicle
ControlPRO Harness White Wire:
Pin 13 (RED/WHITE TRACE AT STEERING COLUMN) in vehicle

ControlPRO Interface Output Wires

The following wires need to be connected to your aftermarket radio.

ControlPRO Harness 3.5mm Jack
If your fitting a Alpine, Clarion, JVC, Pioneer, Sony or Stinger Headunit,
Connect the 3.5mm ControlPRO harness jack to aftermarket radio steering wheel control input jack (SWC Input, W/R Input)
ControlPRO Harness Blue/Yellow Wire
If your fitting a JVC or Kenwood Headunit,
Connect the Blue/Yellow ControlPRO harness wire to the Blue/Yellow wire on the JVC/Kenwood Headunit (SW Control)

4. Programming your interface to your vehicle.

Now that the interface harness is wired, connect the ControlPRO interface.
As the FORD FALCON 2009 - 2011 (FG) requires manual programming with the ControlPRO Interface,
we need to program the vehicle buttons into the interface.

1. TURN ON THE VEHICLE
Start the vehicle, the LED on the interface will light up RED to tell us the product has powered on. If you do not have a RED LED as shown, please move to Step 5 - Troubleshooting.
2. HOLD BLUE PROGRAM BUTTON
Hold down the blue button on the interface till the LED changes to green as pictured.
3. PROGRAM VEHICLE SWC AUDIO BUTTONS
Follow the programming table below in order of the buttons you wish to configure matching the brand of the aftermarket radio being installed.
If vehicle does not have a button, press blue interface button to skip. (ALLOW 1 SECOND BETWEEN SWC BUTTON PRESSES)

Once all vehicle buttons have been pressed. Wait 7 Seconds and the interface LED will flash to confirm buttons have been saved. If interface flashes RED + AMBER, please see Troubleshooing Section 5 (Error code E-1)
4. CONFIRM FEATURES
Confirm each steering wheel audio button operates its feature correctly. If it does not, go back to step 3 above and program buttons in order again.
If each button operates correctly you have completed installation of your ControlPRO Interface.

5. Troubleshooting Problems

Sometimes issues can arise with installation and programming any product. Don't worry we are here to help.

5-1. Start the vehicle.

5-2. Check LED indicator at top of ControlPRO interface. See table below.

ControlPRO LED Illumination Chart

INTERFACE LED COLOR NORMAL INTERFACE OPERATION WHEN MANUAL PROGRAMMING OR REMAPPING INTERFACE
Solid Red Interface is powered on Waiting for you to release the steering wheel button
Solid Green Waiting on steering wheel button to be pressed
Solid amber Firmware Issue. Contact Stinger Australia
Flashing Green Steering wheel button is being pressed in vehicle
Flashing Red+Amber Module ERROR see error list below E-1 Long button steering wheel button press detected
Flashing Green+Red Aftermarket radio DIP switch settings incorrect see error list below E-2
Flashing Green+Amber+Red SWC buttons value being programmed are too close. See error list E-3
ERROR NUMBER POSSIBLE ISSUES THINGS TO TRY
E-1 Vehicle voltage too low Start Vehicle, then try again
Bad ground/earth Make sure ground signal to interface is solid. Try grounding additionally to chassis of vehicle
One button being programmed to two features Interface is seeing same button value twice. Check programming feature list.
Faulty vehicle SWC audio buttons Interface is seeing same button value twice. Only program one button and see if issue is still there. If not continue till find faulty button.
SWC Audio buttons being pressed too quickly Interface is seeing same button value twice. Press each button for 1 second and give 1 second between each button.
E-2 Interface DIP switches set wrong Double check that DIP switches are set to instructions above. Maybe looking at module upside down.
E-3 SWC Audio Buttons being programmed too quickly Make sure to give 1 second between button presses and hold each button for 1 second when programing.
E-4 LED on interface operates correctly, but headunit doesnt respond Some headunits now require the SWC input to be turned on, If the LED flashes on the module when a SWC audio button is pressed, the interface is working correctly. Check your headunit owners manual or contact headunit manufacturer for how to turn on SWC input. Also confirm SWC 3.5mm jack output is connected to radio or blue/yellow trace is hooked to radio harness depending on aftermarket radio brand.
E-5 SWC Audio buttons only work when car's engine isnt running Program interface while vehicle engine is running.
